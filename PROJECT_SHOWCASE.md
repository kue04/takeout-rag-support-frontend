# 外卖客服 RAG 前端工作台项目展示

## 项目定位

这是外卖客服 RAG 系统的 React 前端工作台。它把用户下单、订单售后、客服对话、RAG 诊断、反馈采集和知识运营整合在一个可演示界面里，用来说明 AI 客服系统如何从 Demo 走向可运营应用。

## 主要模块

- 外卖业务模拟：门店、购物车、订单和售后入口。
- 客服对话：围绕订单问题发起 RAG 问答。
- RAG 诊断：展示检索结果、Prompt 预览、证据和诊断字段。
- 反馈闭环：提交有帮助/无帮助反馈，导出 eval case。
- 知识运营：新增、修改、审核、发布和回滚知识条目。
- 项目展示：面向面试的系统介绍页。

## 技术栈

- React 19
- TypeScript
- Vite
- Tailwind CSS
- GSAP
- lucide-react

## 面试亮点

1. 前端不只是聊天 UI，而是把 RAG 链路调试能力产品化。
2. 通过知识运营页面展示“知识变更如何进入 RAG”的真实工作流。
3. 与后端评测、反馈、诊断接口联动，体现 AI 应用闭环意识。

## 本地运行

```powershell
npm install
npm run dev
```

顶部导航点击“项目展示”即可查看面试版介绍。
