# Design

## Visual Direction

真实外卖平台原型，结合订单售后和 RAG 客服解释。主界面应更像美团外卖用户端和淘宝订单页的混合体：搜索、店铺、菜品、购物车、订单、客服入口清楚可见。

## Layout

桌面端：

- 顶部平台导航：位置、搜索、订单、购物车、后端状态。
- 主区域：根据当前视图展示首页、店铺详情、订单列表、订单详情或客服会话。
- 客服视图：左侧订单上下文，中间聊天，右侧 RAG 解释侧栏。

移动端：

- 顶部轻导航和底部导航并存。
- 首页、店铺、订单、客服单列展示。
- RAG 解释作为可展开抽屉。

## Color

浅灰页面背景，白色内容面板。绿色用于主动作和配送状态，橙色用于价格、优惠和注意信息，红色用于错误和风险。不要使用大面积高饱和背景。

## Typography

使用 Geist 风格的系统 sans 字体栈。产品页面使用固定字号，不用过大的营销标题。价格、订单状态、按钮文案要清晰直接。

## Components

- Platform header：位置、搜索、订单入口、购物车数量、RAG 状态。
- Store card：封面、店名、评分、配送时间、起送价、标签。
- Dish card：图片、名称、描述、销量、价格、加购/步进器。
- Cart bar：商品数、总价、配送费、去结算。
- Order timeline：已支付、商家备餐、骑手配送、已送达。
- Support chat：订单问题快捷按钮、用户消息、客服回复、复制回复。
- RAG drawer/sidebar：检索证据、分数拆解、prompt context、trace、复制报告。

## Motion

使用 GSAP 做产品型动效：

- 视图切换 180-260ms 淡入和轻位移。
- 菜品加购时购物车按钮轻微 pulse。
- 订单进度条进入详情页时逐步 reveal。
- RAG 侧栏打开时 slide/fade。

所有动效需要支持 reduced motion。
